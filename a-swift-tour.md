### Simples Values
> Use let to make a constant and var to make a variable.
> https://docs.swift.org/swift-book/documentation/the-swift-programming-language/guidedtour#Simple-Values

```
let CONSTANT = 40
var VARIABLE = 23

print(CONSTANT)
print(VARIABLE)

VARIABLE = 25
print(VARIABLE)

# The following returns a runtime error
CONSTANT = 20
```
